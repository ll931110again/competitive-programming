The grader will then use $$$a$$$ as the initial element in the set $$$S$$$, that is, initially $$$S = \{a\}$$$.</p><p>Next, you can make at most $$$n+3$$$ queries of the following two types:</p><ol> <li> Select an integer $$$x$$$ such that $$$0\leq x\le 2^n-1$$$. The jury will insert $$$f(x)$$$ into $$$S$$$ and then respond with $$$|S|$$$ (i. e., the size of $$$S$$$ after the insertion); </li><li> Select an integer $$$y$$$ such that $$$0\leq y\le 2^n-1$$$. The jury will respond with the number of integers $$$z$$$ such that $$$z\in S$$$ and $$$z\geq y$$$. </li></ol><p>The definition of $$$f(x)$$$ is as follows:</p><p>$$$$$$ f(x)= \begin{cases} x\,\&amp;\,c &amp; \text{if}\, k=1,\\ x\,|\,c &amp; \text{if}\, k=2,\\ x\oplus c &amp; \text{if}\, k=3.\\ \end{cases} $$$$$$</p><p>Here,  $$$\&amp;$$$ denotes the <a href="https://en.wikipedia.org/wiki/Bitwise_operation#AND">bitwise AND operation</a>,  $$$|$$$ denotes the <a href="https://en.wikipedia.org/wiki/Bitwise_operation#OR">bitwise OR operation</a>, and  $$$\oplus$$$ denotes the <a href="https://en.wikipedia.org/wiki/Bitwise_operation#XOR">bitwise XOR operation</a>.</p><p>Your task is to determine the values of $$$k$$$ and $$$c$$$ using at most $$$n+3$$$ interactions.</p><p><span class="tex-font-style-bf">Please note that reporting the answer does not count towards the $$$n+3$$$ interactions.</span></p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>The first line of each test case contains a single integer $$$n$$$ ($$$2 \leq n \leq 60$$$). Then, interaction follows.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$10^5$$$. </p></div><div><div class="section-title">Interaction</div><p>First, you need to output an integer $$$a$$$ ($$$0\leq a\le 2^n-1$$$) — the initial element you chose for set $$$S$$$.</p><p>To make an interaction, output a single line in the following format:</p><ol><li> $$$\mathtt{I}\;x$$$ ($$$0\leq x\le 2^n-1$$$) — Insert $$$f(x)$$$ into set $$$S$$$. After that, the jury will return $$$|S|$$$.</li><li> $$$\mathtt{Q}\;y$$$ ($$$0\leq y\le 2^n-1$$$) — Query how many $$$z$$$-s in set $$$S$$$ satisfy $$$z \geq y$$$. The jury will return the count of such $$$z$$$-s.</li></ol><p>To give the final answer, output a single line in the following format:</p><ul> <li> $$$\mathtt{A}\;k\;c$$$ ($$$1\leq k\le 3$$$, $$$1\leq c\le 2^n-1$$$) — the values of $$$k$$$ and $$$c$$$. </li></ul><p>After printing each query do not forget to output the end of line and flush$$$^{\text{∗}}$$$ the output. Otherwise, you will get <span class="tex-font-style-tt">Idleness limit exceeded</span> verdict.</p><p>If, at any interaction step, you read $$$-1$$$ instead of valid data, your solution must exit immediately. This means that your solution will receive <span class="tex-font-style-tt">Wrong answer</span> because of an invalid query or any other mistake. Failing to exit can result in an arbitrary verdict because your solution will continue to read from a closed stream. </p><p>Note that the grader is <span class="tex-font-style-bf">non-adaptive</span>. This means that the values of $$$k$$$ and $$$c$$$ are fixed at the beginning and will not change based on your interactions.</p><p><span class="tex-font-style-bf">Hacks</span></p><p>To hack, use the following format.</p><p>The first line should contain a single integer $$$t$$$ ($$$1 \leq t \leq 10^4$$$) — the number of test cases.</p><p>The only line of each test case should contain three integers $$$n$$$, $$$k$$$, and $$$c$$$ ($$$2 \leq n \leq 60$$$, $$$1\leq k\le 3$$$, $$$1\leq c\le 2^n-1$$$).</p><p>The sum of $$$n$$$ over all test cases should not exceed $$$10^5$$$.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$To flush, use: </p><ul> <li> <span class="tex-font-style-tt">fflush(stdout)</span> or <span class="tex-font-style-tt">cout.flush()</span> in C++; </li><li> <span class="tex-font-style-tt">sys.stdout.flush()</span> in Python; </li><li> see the documentation for other languages. </li></ul></div></div><div class="sample-tests"><div class="section-title">Example</div><div class="sample-test"><div class="input"><div class="title">Input</div><pre>
